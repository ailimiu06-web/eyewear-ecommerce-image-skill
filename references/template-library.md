# Template Library

This skill can use reusable visual references for eyewear ecommerce production.

The public repository does not include reference images or template screenshots. Add your own rights-cleared assets under `assets/templates/` before using asset-specific template routes.

Use this file when the user asks to use the built-in templates, upload new templates, learn template layout style, or choose a visual direction for sunglasses/eyeglasses detail pages.

Do not copy exact text, model identity, brand marks, logos, watermarks, or unverified claims from any reference image. Learn layout rhythm, section sequencing, product storytelling, and visual hierarchy.

## Template Selection Guide

Choose a template route based on product and sales goal:

- **Clean marketplace route**: use templates 002, 004, 008, 011, 012 for parameter-heavy pages, polarized lens explanation, and clear product grids.
- **Soft fashion route**: use templates 001, 003, 006, 010 for hand-held lifestyle photos, beige editorial tone, model wearing shots, and boutique-style pacing.
- **Color matrix route**: use templates 002, 007, 008, 009, 012 for many color variants or multi-color product families.
- **Technical function route**: use templates 004, 008, 012, 013 for polarized lens tests, UV/outdoor scenes, structural details, and feature proof sections.
- **Handheld try-on route**: use templates 003, 005, 006, 010 for repeated hand-held wearing/try-on photos with color comparisons.
- **Dark premium route**: use template 013 for masculine, black/gold, premium, or performance sunglasses.
- **Children's eyewear route**: use templates 014-051 for children's blue-light glasses, silicone optical frames, myopia-control frames, kids sunglasses, swimming goggles, cycling goggles, basketball goggles, and outdoor protective eyewear.

## Common Layout Patterns Across The Library

Use these patterns when creating new ecommerce image packages:

1. Hero first: model wearing, product still life, or dramatic product closeup.
2. Trust early: icons, short benefit row, or lens-function proof near the top.
3. Detail cards: hinge, temple, nose pad, frame material, lens edge, and logo.
4. Function proof: outdoor comparison, lens test, glare reduction, or travel scene.
5. Parameter clarity: dimensions, material, lens function, weight, and package list.
6. Color browsing: color matrix or long product display ladder.
7. Closing confidence: packaging, care guidance, brand/about, or final product grid.
8. Children's trust loop: audience scene, comfort/safety proof, size/fit explanation, parent-friendly color grid.

## Adaptation Rules

- Match the template route to the product's market position before generating.
- Use the template for structure and pacing, not exact copying.
- Replace all text with copy grounded in user-provided product data.
- Keep product silhouette, lens color, temple shape, and logo faithful to user-provided product views.
- Do not borrow model identity, brand marks, or unsupported feature claims.
- For parameter-heavy pages, prefer clean marketplace or technical templates.
- For fashion pages, prefer model/editorial or hand-held templates.
- For many color variants, prefer color ladder and color matrix templates.
- For children's pages, prefer templates 014-051 and keep all safety, UV, anti-blue-light, anti-fog, impact, and myopia-control claims tied to user-provided sources.
